# 0x02. Python - import & modules

This project focuses on understanding how to import functions, create modules, and use them in Python programs.

## Requirements
- All files are interpreted/compiled on Ubuntu 20.04 LTS using Python 3.8.5
- The first line of every file should be `#!/usr/bin/python3`
- Code should follow pycodestyle (version 2.8.*)
- All files must be executable
